With this API, you can get information on nearby stops a fitted location based on latitude and longitude.

API key

It requires a valid API key passed as parameter "key" in all method calls. An API key is obtained by creating a project using this API. More on how to create and use API keys can be found here . To get an API key, you have to accept APIets  license . There are some limitations on the number of calls per minute and per month.

Note: To receive the Gold level requires that you write a detailed description of the project on Trafiklab. Projects lacking description will be denied the Gold level.

Level Max calls / month Max call / minute
Bronze 10000 30
Silver 500 000 60
Gold Based on needs
Our goal is that everyone who wants the data to have access to this, and that it will remain free of charge. The limitations we have today is because we want to ensure that our technology works to as many as possible. Before upgrading a service like we have a forecast of how much traffic the service will generate in relation to how many travelers will benefit from the information. To get your account upgraded, use the upgrade button on  your key .
